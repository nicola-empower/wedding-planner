# The Ultimate Wedding Planner

**A premium, AI-powered wedding planning experience designed for the modern couple.**

This application represents the future of event planning software. Blending elegant design with cutting-edge technology to simplify the most important day of your life.

##  Key Features

### For the Bride & Groom
A centralised command centre to manage every detail without the stress.
-   **Dashboard Overview**: Real-time tracking of budget, guest counts, and tasks.
-   **Smart Vendor Management**: Keep track of photographers, caterers, and florists with ratings and cost analysis.
-   **AI-Powered Design**: **Exclusive Feature!** Use our integrated **Gemini AI** to generate unique, harmonious colour palettes for your bridesmaids and decor instantly.
-   **Theme Customisation**: **New!** Personalise your planning experience with our dynamic theme switcher. Choose from **Sage**, **Ocean**, **Rose**, or **Lavender** to match your wedding vibe.
-   **Beauty & Style**: Manage appointments for hair, makeup, and nails.
-   **Guest List**: Comprehensive RSVP tracking, dietary requirements, and table planning.

### For the Guests
A seamless, mobile-first experience that keeps your loved ones informed.
-   **Digital Concierge**: Instant access to the Order of the Day, venue locations, and timings.
-   **Interactive Menu**: Mouth-watering preview of the wedding breakfast and evening buffet.
-   **Live Seating Plan**: Guests can easily find their seats with our searchable table plan.
-   **Shared Gallery**: A communal space for guests to view and upload memories.

##  Technical Stack

Built with the latest web technologies to ensure performance, scalability, and a premium user experience.

-   **Framework**: [Next.js 14](https://nextjs.org/) (App Router & Server Components)
-   **Language**: [TypeScript](https://www.typescriptlang.org/) for robust, type-safe code.
-   **Styling**: [Tailwind CSS v4](https://tailwindcss.com/) - Utilising the latest features for high-performance, utility-first styling.
-   **AI Integration**: [Google Gemini API](https://deepmind.google/technologies/gemini/) for generative design features.
-   **UI Components**: Custom-built accessible components using [Lucide React](https://lucide.dev/) icons and [Radix UI](https://www.radix-ui.com/).
-   **Animation**: Smooth, native-feeling transitions for a polished feel.

##  Getting Started

This is a developer-ready demo. To run it locally:

1.  **Clone the repository**
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Set up Environment**:
    Create a `.env.local` file and add your Google Gemini API key:
    ```bash
    GOOGLE_API_KEY=your_api_key_here
    ```
4.  **Run the development server**:
    ```bash
    npm run dev
    ```
5.  Open [http://localhost:3000](http://localhost:3000) to see the magic.
